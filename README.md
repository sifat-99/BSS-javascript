## How to Run the Files

Ensure you have [Node.js](https://nodejs.org/) installed on your machine.

### Run a single file:
Navigate to the root directory and run:
```bash
node "Folder Name/filename.js"
```
Example:
```bash
node "Control flow/fizzbuzz.js"
```
